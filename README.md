# RC Finish Analyzer Desktop

## 起動

```sh
./run_desktop.sh
```

## 現在の実装範囲

- 録画動画を開くと自動で解析開始
- スタートライン、ゴールラインの2点指定
- 真横 / 車両の右前 / 左前 / 右後ろ / 左後ろのカメラ位置選択
- 単純ゴール順モード / 周回モードの切替
- 動画読込後に自動解析開始
- OpenCV差分によるゴール通過候補検出
- OpenCV候補は通知・順位表示せず、YOLO投入のトリガーとしてだけ使う
- YOLOで車両として確定した時だけ順位一覧へ表示
- YOLO確定時に検出枠付きの切り抜き画像を表示
- `close_finish` 判定
- 順位一覧と僅差レビュー表示
- YOLO26 `n / s / m` のプロファイル選択UI
- OpenCV候補検出後、上限付きキューでYOLOをバックグラウンド実行
- YOLO候補キューは最大3件
- 確定イベントは最大500件

## 注意

- 画像はメモリに保持し続けず、一時ディレクトリへ保存する
- 通常イベントは切り抜き画像だけを一時ROMへ保存する
- 僅差イベントだけレビュー画像を縮小して一時ROMへ保存する
- レースリセット時とソフト終了時に一時ファイルを削除する
- `yolo26n.pt` は取得済み
- `yolo26s.pt` と `yolo26m.pt` は初回選択時に取得される可能性がある
- 本番前に実動画でしきい値とライン位置の調整が必要
